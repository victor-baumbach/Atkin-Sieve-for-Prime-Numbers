## What is this?
My attempt at writing an Atkin Sieve in Python using what I understood from reading the [Wikipedia page on the sieve](https://en.wikipedia.org/wiki/Sieve_of_Atkin).

## What did you use?
I wrote this using Github Codespaces, hoping to understand the limits and usecases such a development environment.
Originally my solution used the `floor()` and `sqrt()` standard methods from the `math` module, but I removed them suspecting that people would wrinkle their noses at the use of a square root function in a prime number generator.
